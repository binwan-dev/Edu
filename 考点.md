### 单元考试一
### 单元考试二
#### 市场营销的含义
1985年美国市场营销协会对市场营销所下的定义是：“市场营销是（个人和组织）对思想、产品和服务的构思、定价、促销和分销的计划和执行过程，以创造达到个人和组织的目标的交换”
#### 企业选择目标市场所采用的战略
企业选择其目标市场可以采用三种战略，无差异市场营销、差异市场营销、集中市场营销
#### 影响企业外部定价的因素
从外部来看，主要有：市场需求，市场结构，国家政策及其他外部因素
#### 消费品市场最重要的促销方式
消费品市场上，最重要的促销方式是广告，人员推销则是产业市场最重要的促销工具。
#### 网上市场间接调查工具
1. 利用搜索引擎收集
2. 利用公告栏收集  
3. 利用新闻组收集  
4. 利用电子邮件收集
#### 效率与效益的关系
效率是指以正确的方式工作，强调作业方法的有效性与合理性；效益是指从事正确的工作，强调所从事的业务本身的合理性与方向的正确性。由于企业外部环境、内部条件是动态的，企业的目标也不是一成不变的，因此其效率、效益也是动态的。运作管理应密切监督效率、效益的变化趋向，坚持以长期效益为主，兼顾短期效率的原则，及时采取措施，加强对长期效益、短期效率的管理。
#### JIT（准时生产）
尊重人的作用是JIT发挥作用的关键，JIT鼓励团队合作，强调“少人化“，要求与供应商紧密合作。
#### JIT 生产要点
1. 暴露问题的瓶颈
2. 消除浪费
3. 改进产品设计
4. 改进工厂布局和流程设计
5. 改进人力资源
6. 全面质量管理
7. 拉式调度和看板生产控制系统
8. 与供应商紧密合作
#### 虚拟企业的别称
虚拟企业也称动态联盟，是敏捷制造模式中最基础和核心的组织形式。
#### 敏捷制造的起源地
敏捷制造是由美国针对当时各项技术迅速发展、渗透，国际、国内市场竞争日益激烈的形势，为维护世界第一工业国地位，维持美国人的高生活水平而提出的一种制造生产组织模式和战略计划。
#### 虚拟企业最基本的形式
虚拟生产是虚拟企业最基本的形式，是指企业将其产品的直接生产功能弱化，把生产功能用外包的办法转移到别的企业去完成，而自己只保留最具优势并且附加值高的开发和营销功能，并强化这些部门的组织管理
#### 财务管理
财务管理的目标取决于企业的总目标，并且受企业财务管理本身特点的制约。  
#### 财务计划方法
财务计划方法有：平衡法、比例法、定额法和因素法
#### 信息理财观念
在网络时代，企业财务管理人员必须牢固树立信息理财的观念，从全面、准确、迅速地收集、分析和利用信息入手，进行财务决策和资金运筹
#### 泛财务资源概念
泛财务资源概念是对传统财务资源观念的延拓。从形态上来看，泛财务资源可以分为硬财务资源和软财务资源两大类。其中，硬财务资源是指客观存在的，在一定技术、经济和社会条件下能被企业利用的有形资源，其构成主体是自然资源和传统的财务资源。软财务资源是以智力为基础的资源或无形的资源，包括知识资源和时间资源。
#### 网络经济时代企业的主要生产要素与价值决定因素
在网络时代，知识是利润形成的主要来源，是企业的主要生产要素与价值决定要素，它理所当然地成为分配的主要依据。 
#### 网络时代企业财务管理的目标
网络时代企业财务管理的目标是企业价值最大化 
#### 网络财务管理的介质化
网络财务趋于电子化
#### 培训工作流程阶段
1. 培训需求分析
2. 培训设计和实施  
3. 培训评估
#### 绩效薪资的含义
绩效薪资是根据员工年度工作绩效评价的结果而确定的对基础薪资增加的部分，因此它是员工的优良工作绩效的一种奖励。
#### 影响薪酬最重要的因素
薪酬制定者直接决定了员工所能得到的薪水，是影响薪酬最重要的因素
#### 企业财务管理原则
1. 系统原则  
2. 平衡原则  
3. 弹性原则  
4. 比例原则  
5. 灵活性原则  
6. 优化原则
#### 企业融资的主要方式
1. 吸收直接投资  
2. 发行股票  
3. 发行债券  
4. 银行借款  
5. 商业信用  
6. 租赁融资  
7. 内部积累
#### 人力资源的特征
1. 双重性  
2. 能动性  
3. 实效性  
4. 再生性  
5. 社会性
#### 市场营销环境分析
市场营销环境分析包括微观环境分析和宏观环境分析。
1. 微观环境分析
   是指那些对企业服务其顾客的能力构成直接影响的各种力量，包括 企业自身、市场营销渠道企业、市场、公众、竞争者
2. 宏观环境分析
   是指那些能够对企业产生市场机会和环境威胁的社会环境，包括 人口环境、自然环境、经济环境、技术环境、政治法律环境、社会文化环境
#### 财务管理方法
1. 财务预测方法
   1. 定性预测法
   2. 定量预测法
2. 财务决策方法
   1. 优化对比法
   2. 数学微分法
   3. 线性规划法
   4. 概率决策法
   5. 损益决策法
3. 财务计划方法
   1. 平衡法
   2. 比例法
   3. 因素法
   4. 定额法
4. 财务分析方法
   1. 对比分析法
   2. 比率分析法
   3. 综合分析法
#### 网络财务管理的特点
1. 网络财务拓展了财务管理的空间
2. 网络财务加快了财务管理的时效
3. 全新的运作方式提升了财务管理的效能
#### 人力资源管理概念和特点
人力资源是指运用科学的管理方法，在企业战略的指导下，以人力资源战略、规划和工作分析为基础，对人力资源进行获取与配置、开发与培训、考核与激励、规范与约束、安全与保障、凝聚与整合，最终实现企业目标和员工价值的过程

特点：
1. 综合效率原则
2. 使用和开发相结合原则
3. 公平竞争原则
#### 绩效管理
绩效管理是指为了达到组织目标，通过持续开放的沟通，形成组织所期望的利益与产出，并推动团队和个人做出有利于目标达成的行为，即通过持续的沟通和规范化的管理不断提高员工和和组织绩效、提高员工能力和素质的过程
#### 虚拟团队
虚拟团队是为了开发某种特殊的产品或解决某一特殊的问题而设置的团队，是信息时代的一种新的组织模式，其成员可以跨越时间、空间甚至是组织边界，借助于现代信息通信技术进行交流和沟通。虚拟团队的成员一般是来自不同领域的专家组成，他们利用自己的专业知识相互协作，一般实行自我管理和自我指导。就时间而言，可以是短暂的，也可以是长期的。

虚拟团队的人力资源管理可以考虑从以下几方面着手：
1. 选拔有相关能力的领导。
2. 要有高素质的团队成员
3. 建立以“信任”为基础的团队文化
4. 提高沟通效率
5. 采用适合虚拟团队特点的激励方式
   
#### 网络营销站点的功能
企业开展网络营销首先必须要建立自己的网络站点。根据企业站点建立的目标的不同，企业网站可以分为五种类型：信息型、广告型、信息订阅型、在线销售型、技术支持服务型。
一般来说网络营销站点功能有：
1. 企业信息发布
2. 信息交流沟通
3. 网上销售
4. 售后服务
5. 个性化服务

#### 网络广告方式
1. 用自己的网站发布广告
2. 在他人网站发布广告
3. 使用旗帜广告交换服务网络
4. 在网上报纸或杂志上发布广告
5. 电子邮件发布广告
6. 在虚拟社区和公告栏上发布广告
7. 使用新闻组发布广告  

### 单元考试三
#### 知识
知识是指经过人的思维整理过的数据、信息、形象、意象、价值标准以及社会其他符号化产物等。信息是经过加工组织后的数据，而知识是基于信息之上的有关事实之间的因果或相关性的联系，它可以帮助我们预测未来的结果
#### 原理知识
原理知识，是指“知道为什么”的知识，是关于事物的原理和客观规律方面的知识，通常属于科学范畴。
#### 知识管理的原则
实施企业知识管理需要把握积累、信任、交流、共享和创新的原则。
#### 个人化策略
个人化策略主要采用的是“专家”模式。这种策略适合致力于为客户处理没有明确答案的特殊化问题，提供非常人性化的服务或生产个性化产品的企业。

#### 供应链管理
供应链管理是以顾客为导向，在供应链中，顾客是源头，是指挥者，而不是供应链产品或服务的被动接收者。供应链从顾客开始，到顾客结束。
#### 供应链管理的含义
1. 供应链管理的目标是为了给顾客提供满意的产品或服务
2. 供应链管理是一种集成管理模式，供应链管理把不同的企业集成起来提高整个供应链的效率。
3. 供应链管理强调提高整个供应链的竞争能力
4. 供应链管理最终为供应链中所有成员带来价值增值
5. 供应链管理的目标是通过对供应链的优化配置来实现的
#### 7R 原则
供应链管理的本质目标是“7R”原则，即将合适的产品在合适的时间、合适的数量、合适的质量和合适的状态送到合适的顾客的合适的地点，并使总成本最小或受益最大
#### 业务外包的含义
业务外包是指在供应链管理环境下，企业把主要精力放在关键业务上，而把非核心业务交由合作企业完成，从而获得比单纯利用内部资源更多的优势。
#### 企业与竞争对手相区别的最根本标志
企业文化是社会文化与组织管理实践相融合的产物，是企业的灵魂，良好的企业文化是企业生存和发展的源动力，
#### 人本文化
网络时代，企业的发展关键在于信息、知识和创造性，而这些资源的唯一来源是人，人是决定管理绩效及整个企业效益最根本性的因素
#### 企业文化的特点
企业文化的特点有：民族性、客观性、独特性、科学性、动态性、约束性和辐射性
#### 知识的分类
根据经合组织对知识的划分，可以将“知识”归纳为四种类型：事实知识、原理知识、技能知识、人力知识
#### 供应链采购的特点
与传统的采购相比，供应链采购具有以下特点
1. 行业特征
2. 管理者素质
3. 已有企业文化模式
4. 追求独特模式
5. 形成的长期性
#### 知识的概念及特点
知识是指经过人的思维整理过的数据、信息、形象、意象、价值标准以及社会的其他符号化产物等。它不仅包括科学技术知识、还包括人文社会科学知识，商业活动、日常生活和工作中的经验知识，人们获取、运用和创造知识的知识，以及面临问题做出判断和解决方法的知识
知识的特点：
1. 不可替代性
2. 互补性
3. 不可逆性
4. 无形性
5. 非稀缺性
6. 非消耗性
7. 可共享性
8. 增值性
#### 供应链管理和物流管理的区别
两者主要区别在：
1. 物流涉及原材料、零部件在企业间流动，而不涉及生产制造过程的活动
2. 供应链管理包括物流活动和生产制造活动
3. 供应链管理涉及从原材料到交付给最终用户的整个物流增值过程，物流涉及企业间的衔接管理活动
#### 供应链管理环境下物流管理的特点
1. 提高了物流系统的快速反应能力
2. 降低了物流系统成本
3. 增进了物流系统的无缝连接
4. 物流服务方式的多样化
5. 提高了顾客的满意度
#### 企业文化的构成
1. 企业哲学
2. 企业价值观
3. 企业精神
4. 企业制度
5. 企业道德规范
6. 企业形象
#### 网络时代企业文化的发展体现方面
1. 倡导人本文化
2. 倡导合作文化
3. 倡导创新文化
4. 倡导学习文化
5. 倡导速度文化
6. 倡导融合文化
7. 倡导生态文化
8. 倡导形象文化
#### 企业文化管理制度包括的方面
企业文化的管理制度是为了更好的实现企业文化管理目标而制定的，一般来说，包含以下几个方面：
1. 文化意义符号管理制度
2. 企业文化沟通管理制度
3. 企业文化氛围管理制度
4. 企业文化的遗传管理制度
#### 知识管理策略的选择应注意的问题
1. 企业知识管理策略应反应其采取的总体战略
2. 知识管理策略的选择应考虑企业的发展阶段
3. 知识管理策略的选择应考虑企业事业部之间的关系
4. 等待添加
5. 知识管理策略的选择应考虑产品所处的生命周期
6. 知识管理策略的选择与员工所依靠的知识特性相关
#### 企业文化的内涵
所谓的企业文化，是指在一定的社会经济条件下，企业在长期的经营过程中逐渐形成并为全体成员遵循的共同意识、价值观念、职业道德、行为规范和准则的总和。它是企业的个性、素质、目标和氛围，也是一个企业或组织在自身发展过程中形成的以价值为核心的独特的文化管理模式。
企业文化是社会文化和组织管理实践相融合的产物，是企业的灵魂，良好的企业文化是企业存在和发展的源动力，是区别于竞争对手的最根本标志。
#### 企业文化的作用
1. 导向作用
2. 凝聚作用
3. 规范作用
4. 激励作用
5. 沟通作用
6. 绩效作用
